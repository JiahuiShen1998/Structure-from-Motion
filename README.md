# Structure from Motion

This repository contains a cleaned and presentation-ready version of our team project on Structure from Motion (SfM), focusing on camera calibration, image undistortion, feature matching, and geometry estimation for 3D scene reconstruction.

## Project Overview

Structure from Motion is a classical computer vision pipeline that reconstructs scene geometry and estimates camera motion from multiple views.  
This project covers key stages of the SfM workflow, including:

- Camera calibration
- Image undistortion
- Feature extraction and matching
- Estimation of scene geometry and camera motion
- Intermediate visualization and reconstruction-related processing

This repository was cleaned and reorganized for documentation and portfolio purposes.
